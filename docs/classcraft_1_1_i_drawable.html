virtual void craft::IDrawable::calcPixel ( int16_t  rx,
int16_t  ry 
)
inlinevirtual

Calculate the pixel to be rendered at the current position.

The coordinates, rx and ry, are in local coordinates. rx will always be n + 1 higher than rx the previous call to calcPixel, calcMaskPixel or skipPixel for a given line (prepareLine is called at the start of a line). ry will be identical.

The renderer expects the results of calcPixel to be in _rc (the color, 24bit RGB) and _ra (alpha 0.0 - 1.0).

Parameters
rxThe current X position in local coordinates
ryThe current Y position in local coordinates

Reimplemented in craft::Gradient, craft::Shape, craft::Sprite, craft::PlySprite, craft::Text, craft::Box, and craft::LinearGradient.
