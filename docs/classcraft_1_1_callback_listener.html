Detailed Description

Listener that fires a callback. Example usage:

// Define the callback function boolean handleTheEvent( uint32_t event, void* data ) { // Do something Serial.printf("Event received. The event is %l.\n", event );

// Continue to listen to events return true; }

// Create the listener CallbackListener* listener = new CallbackListener( handleTheEvent );

// Add the listener to listen for the "action_play" event messenger->addListener( Event::action_play, listener );
