void craft::LineBuffer::flip ( bool  flush = false)

Advance the buffer to the next line and flush to hardware if necessary The buffer is advanced to the next line. If the buffer is filled or the end of the drawing region is reached, the buffers are flipped and the pixels are flushed to the hardware. Setting flush to true will also flip and flush the buffer, even if it is not full.

Parameters
flushIf true will flush the buffer to the hardware even if not full
