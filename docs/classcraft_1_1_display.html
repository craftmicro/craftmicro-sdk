Member Data Documentation

◆ framebuffer

void* craft::Display::framebuffer = nullptr

Pointer to the framebuffer.

The framebuffer must be the size of the entire display, and the type must match the pixelFormat property. If pixelFormat is RGB565, the framebuffer must be an array of uint16_t[width * height] If pixelFormat is RGB888, the framebuffer must be an array of uint8_t[width * height * 3]

If there is no framebuffer, implement drawBegin, drawEnd, and either drawPixel565 or drawPixel888. If a framebuffer is provided these functions do not need to be implemented.
