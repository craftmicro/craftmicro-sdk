Detailed Description

Filter base class A filter is used to modify pixels of a display object just before rendering them.

A filter is treated as a display object. All the render methods of IDrawable are applied to filter (except calcPixel) and can be used to create complex filters. Most filters will not need to use them and will only implement filterPixel.
